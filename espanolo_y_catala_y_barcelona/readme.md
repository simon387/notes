**HOW TO TYPE ACCENTS WITH A SPANISH KEYBOARD**

(and spanish layout by windows system)

**use this ` or ´ or ^ followed by the vocal!**

or you can use this table:

|---|---|---

|Í | <kbd>ALT</kbd> + 0205|
